最近做了一个小项目,有这样一个需求,按照用户输入的顺序从数据库中取出数据。但是默认sql查询会有自动排序的功能,导致查询的结果是按照数字的顺序出现。百度查询后发现解决办法，在此记录一下sql语句写法

```sql
SELECT * FROM 表名 WHERE 关键字 IN(数字1, 数字2, 数字3, 数字4, 数字5)  ORDER BY INSTR(',数字1, 数字2, 数字3, 数字4, 数字5,',CONCAT(',',关键字,','))
```
