# ssh实现免密登录

### 第一步：

本地生成ssh 公钥及私钥

(如果已有ssh公钥则忽略此步骤)

```bash
ssh-keygen -t rsa 
```

### 第二步：

将本地生成的id_rsa.pub 的内容写入服务器的 用户名/.ssh/authorized_keys文件中，没有可以新建此文件

### 第三步：

安全起见，将服务器.ssh/文件夹权限改为700，authorized_keys文件权限改为600（or 644）
